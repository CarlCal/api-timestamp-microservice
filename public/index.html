Timestamp microservice

API Basejump: Timestamp

Description:

By typing in a unix timestamp (seconds) or a natural langauge date, you will receive a JSON containing the unix and natural variation of the parameter you gave. If the parameter is neither a date nor a unix stamp it will return the same JSON with both values being null.

Note: This explains the two numbers after the procentage sign in the URL


Usage:

https://carlcal-timestamp-ms.herokuapp.com/december%2515,%252016

https://carlcal-timestamp-ms.herokuapp.com/1481817600

Output:

{ "unix": 1481817600, "natural": "December 15, 2016" }
